# Shivakumar C — Portfolio Website

## 📁 Folder Structure

```
shivakumar-portfolio/
│
├── index.html          ← Main HTML (all sections)
│
├── css/
│   └── style.css       ← All custom styles
│
├── js/
│   └── main.js         ← Animations, scroll, form logic
│
└── assets/             ← (Add your photo here as "profile.jpg")
```

## 🚀 Sections

| Section   | ID         | Description                          |
|-----------|------------|--------------------------------------|
| Home      | `#home`    | Hero, name, role, CTA buttons        |
| About     | `#about`   | Services, bio, stats, skill bars     |
| Projects  | `#projects`| 3 project cards with tech tags       |
| Contact   | `#contact` | Info + contact form                  |

## 🛠 Tech Used
- HTML5
- CSS3 (custom variables + animations)
- Bootstrap 5.3
- Tailwind CSS utilities (via CDN class usage)
- Vanilla JavaScript
- Font Awesome 6 (icons)
- Google Fonts: Syne + Space Grotesk

## 📌 How to Run
Just open `index.html` in any browser — no build step needed!

## 💡 To Customize
- Replace `SK` initials in `.hero-avatar` with your actual photo
- Update GitHub / LinkedIn URLs in the Contact section
- Add your project live links

## 🎨 Theme
Dark theme with coral/orange accent (#ff5c35), inspired by the Jensen Omega portfolio reference.
